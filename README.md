# CRUD Frontend Application (React)

This project is the frontend interface for a full-stack CRUD system, built using **React** and connected to a **Node.js + Express + MySQL backend**.

Users can create, read, update, and delete user records with a responsive and user-friendly UI.

---

## Features

- Add User
- Edit User
- Delete User
- View Users
- Connected to REST API
- Responsive UI
- Built using React Hooks & Axios

---

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| React JS | Frontend UI |
| Axios | API Communication |
| Bootstrap / CSS | Styling |
| React Hooks | State Management |
